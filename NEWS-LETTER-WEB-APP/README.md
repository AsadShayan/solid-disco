# NEWS-LETTER-WEB-APP
A news-letter web app with frontend , backend and a (localhost) server

# Tools Incorporated in the Project :
1. NodeJS
2. ExpressJS
3. Bootstrap
4. HTML
5. CSS
6. git (for version control)
7. MailChimp API (that controls the backend mostly)
